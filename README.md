# 🧛 VampireApp - App for TTRPG club

Modern Android application with Telegram-based authentication and backend integration.

⚠️ Project in active development.

---

## 🚀 About the Project

VampireApp is an Android application built with a focus on clean architecture.

The project currently implements:

- Telegram Login integration
- Backend verification
- Firebase custom token authentication
- Secure login flow inside Custom Tabs
- Clean architecture setup

The UI layer is under development.

---

## 🔐 Authentication Flow

The app implements a custom Telegram authentication flow:

1. Telegram Login Widget
2. JS Interface communication
3. Backend verification
4. Firebase custom token generation
